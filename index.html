本地运行指南

# 1. 安装依赖

pip install flask flask-cors gpxpy requests werkzeug

# 2. 配置Kimi API Key（可选，不配置则使用本地算法）

export KIMI_API_KEY="your-api-key-here"

# 3. 启动服务

python app.py

# 4. 浏览器访问

http://localhost:5000
